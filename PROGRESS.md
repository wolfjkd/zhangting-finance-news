## 四、技术架构

### 后端（Python）
- **框架**：pywebview + WebView2
- **核心模块**：
  - `app.py` - 主程序入口，WebSocket 代理，行情 API
  - `data_source_manager.py` - 多数据源管理器
- **网络库**：`requests.Session`（`trust_env=False`，直连国内 API）
- **数据存储**：SQLite（去重数据库）
- **打包**：PyInstaller 单文件 exe

### 前端（HTML/CSS/JS）
- **渲染**：WebView2
- **通信**：`pywebview.api` 调用后端
- **本地存储**：IndexedDB（历史消息、收藏）+ localStorage（配置）
- **核心模块**：
  - `app.js` - 主逻辑
  - `modules/news-classifier.js` - 新闻分类与优先级
  - `modules/stock-quote.js` - 行情查询
  - `modules/history-storage.js` - 历史存储

### 对话框样式规范
- 宽度：`width: 90%; max-width: 420px`
- 圆角：`border-radius: 10px`
- 边框：`1px solid var(--border)`
- 阴影：`0 8px 32px rgba(0,0,0,0.15)`
- 关闭按钮：右上角红叉 SVG
- 通用类：`.dialog-close-btn`

## 七、开发规范速览

### 版本号规则（SemVer）
- 格式：`vMAJOR.MINOR.PATCH`
- 三处同步：README.md + 代码硬编码 + Git Tag/Release
- CHANGELOG.md 每次更新

### 打包规则
- exe 命名：`涨停财经聚合播报_v{版本号}.exe`
- 所有历史版本保留在 dist/
- 打包命令：双击 build.bat

### 代码规范
- Python：`requests.Session(trust_env=False)` 直连国内 API
- JS：通过 `pywebview.api` 调用后端
- CSS：CSS 变量 + 响应式设计
- 调试：`DEBUG_MODE` 开关控制日志
