# 涨停财经聚合播报 v3.8.1（开源版）

轻量级桌面财经新闻推送客户端，基于 pywebview（WebView2 内核），无需浏览器即可实时接收聚合消息推送，并集成了东财公告、券商研报、北向资金等多数据源。

**📢 开源声明：本项目已开源，采用 MIT License，所有功能完全免费开放！**

## 功能

### 核心功能
- **实时新闻推送**：WebSocket 连接，秒级延迟
- **历史消息加载**：启动时自动加载，支持手动加载更多
- **窗口可调**：任意调整长宽高（最小 320×400）
- **窗口置顶**：一键切换置顶/取消置顶
- **自动滚动**：新消息自动滚到顶部，手动滚动后暂停
- **消息卡片**：显示时间、来源、标题、内容摘要、关联股票
- **暗色主题**：深色背景，护眼舒适

### 多数据源（V3.1 新增）
- **涨停财经聚合平台**：主要数据源，WebSocket 实时推送
- **东方财富公告**：`np-anotice-stock.eastmoney.com`，轮询获取重要公告
- **券商研报**：`reportapi.eastmoney.com`，轮询获取研报评级变动
- **北向资金**：`push2delay.eastmoney.com`，轮询获取北向资金实时数据
- **股票行情**：腾讯财经 API（`qt.gtimg.cn`），获取实时行情

### 新闻分类与优先级系统（V3.1 新增）

每条新闻自动分析标题和内容，匹配到以下 **6 个分类标签** 和 **4 个优先级标识**：

#### 分类标签

| 标签 | 图标 | 颜色 | 触发关键词举例 |
|------|------|------|---------------|
| 政策 | 🏛️ | 红 | 央行、财政部、国务院、发改委、降准、加息、MLF、LPR… |
| 资金 | 💰 | 青 | 北向资金、融资融券、主力资金、净买入、大单、机构… |
| 公司 | 🏢 | 蓝 | 业绩、财报、公告、重组、增持、减持、问询函、被ST… |
| 宏观 | 🌍 | 绿 | GDP、CPI、PMI、利率、汇率、通胀、经济… |
| 行业 | 📊 | 黄 | 板块、龙头、涨停、跌停、异动、轮动… |
| 技术指标 | 📈 | 紫 | 突破、支撑、MACD、金叉、死叉、成交量、换手率… |

一条消息可同时匹配多个分类（如"央行降准利好地产"同时显示"政策"和"宏观"）。

#### 优先级标识

| 等级 | 颜色 | 边框 | 触发词举例 |
|------|------|------|-----------|
| CRITICAL | 红 | 红色脉冲 | 央行降准、闪崩、黑天鹅、系统性风险… |
| HIGH | 橙 | 橙色 | 重磅、突发、涨停、跌停、利好、利空… |
| MEDIUM | 黄 | 黄色 | 关注、风险、机会… |
| LOW | — | — | 无关键词命中时不显示（不兜底） |

### 其他功能
- **相似消息去重**：Dice 系数阈值 0.55，多个来源推送同一事件只显示一条
- **关键词高亮 + 警报**：自定义关键词列表，命中时高亮显示，可选警报声和弹窗
- **语音播报**：TTS 合成语音播报新消息标题
- **历史存储**：IndexedDB 本地存储，支持全文搜索和导出
- **新闻详情弹窗**：东方财富公告和券商研报的消息，点击标题可弹窗查看详细内容
- **数据源管理**：可视化管理各数据源的启用/禁用和连接测试
- **赞赏支持**：完全免费开源，欢迎自愿打赏支持开发

## 运行环境

- Windows 10/11（需要 WebView2 Runtime，Win10 21H2+ 自带）
- Python 3.13+

## 安装依赖

```bash
pip install pywebview requests websocket-client
```

**方式一：直接运行 exe（推荐，无需 Python 环境）**
双击 `dist/涨停财经聚合播报_v3.8.1.exe`

**方式二：Python 脚本启动**

## 技术方案

- **后端**：Python (pywebview + requests + websocket-client)
  - `fetch_initial()` — 抓取 724.guzhang.com，提取 token + WS 配置 + 初始新闻 HTML
  - `start_ws()` — 启动 WebSocket 代理线程，实时推送新闻到前端
  - `load_more()` — 加载更早的历史消息
  - `get_stock_quote()` — 腾讯财经 API 实时行情查询
  - 数据源管理器 — 东财公告/研报/北向资金轮询 + 消息回调推送
- **前端**：纯 HTML/CSS/JS，WebView2 渲染
  - 通过 `pywebview.api` 调用 Python 后端（无跨域问题）
  - 解析 HTML 渲染新闻卡片
  - WebSocket 事件通过 `window._onBackendEvent()` 回调
  - 新闻自动分类 + 优先级判断 + 标签渲染
- **数据源**：724.guzhang.com + 东方财富 + 腾讯财经
- **实时推送**：WebSocket（wss://swoole2.guzhang.com:443）
