# 第三方开源组件声明

> **版本**：v1.0  
> **生效日期**：2026-06-29

## 1. 概述

本软件使用了以下第三方开源组件，这些组件均遵循各自的开源许可证协议。

---

## 2. Python 第三方库

| 组件名称 | 版本 | 许可证 | 用途 | 项目地址 |
|---------|------|--------|------|---------|
| pywebview | 5.x | MIT License | 桌面应用框架，提供 WebView2 渲染能力 | https://github.com/r0x0r/pywebview |
| requests | 2.x | Apache 2.0 License | HTTP 请求库，用于获取第三方数据 | https://github.com/psf/requests |
| websocket-client | 1.x | BSD License | WebSocket 客户端，用于实时推送连接 | https://github.com/websocket-client/websocket-client |
| PyInstaller | 6.x | GPL License | 打包工具，将 Python 代码打包为 exe | https://github.com/pyinstaller/pyinstaller |

---

## 3. 前端开源组件

本软件前端使用原生 HTML/CSS/JavaScript，未引入第三方前端框架或库。
