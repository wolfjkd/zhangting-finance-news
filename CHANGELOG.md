# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/lang/zh-CN/).

## [v3.6.0] - 2026-06-28

### Changed
- 自选股对话框布局优化：关闭按钮右上角，添加分组/自选按钮左上角一行排列
- "+添加"按钮改名为"+自选"，与"+分组"按钮大小一致、颜色区分
- 消息流改为卡片式呈现：圆角矩形、小间隔、背景色区分
- 消息来源标签移到内容左下方
- 删除"我的收藏"和"复盘报告"功能（清理残留代码和CSS）
- Windows标题栏暗色模式支持（DWM API + GetAncestor方案）

### Fixed
- 主题切换一致性：所有硬编码颜色改为CSS变量

## [v3.5.1] - 2026-06-28

### Added
- 自选股异动提醒：价格涨跌幅超阈值时弹窗+声音提醒
  - 定时轮询自选股行情（每分钟检查）
  - 可设置阈值和检查间隔
  - 支持声音提醒（Web Audio API）
  - 冷却机制防止重复提醒

### Added
- 自选股分组管理：支持创建/删除/重命名分组
  - 添加分组按钮
  - 分组菜单（⋮按钮）重命名/删除
  - 移动股票到指定分组

### Fixed
- 分组管理CSS样式缺失问题

### Changed
- IndexedDB批量写入优化
  - 新增 `storeMessagesBatch` 方法
  - 消息缓冲机制（10条批量 + 2秒延迟刷新）
  - 减少数据库事务次数，提升性能

## [v3.5.0] - 2026-06-28

### Added
- 自选股管理功能
  - 自选股添加/删除（输入股票代码自动获取名称）
  - 自选股列表管理对话框
  - 自选股数据 IndexedDB 持久化
  - 新闻自动匹配关联自选股（标题+内容扫描）
  - 匹配自选股的新闻高亮标记（左侧橙条 + ⭐自选徽章）
  - 自选股行情快速查看（点击徽章或行情按钮）
  - 工具栏新增"自选股"按钮

## [v3.4.0] - 2026-06-28

### Added
- 复盘报告功能：一天新闻汇总+关键词统计
  - 高频关键词Top 20统计
  - 分类分布、优先级分布、来源分布
  - 时间分布（按小时统计）
  - 支持导出Markdown格式报告
  - Toolbar新增"复盘报告"按钮
  - 支持自定义日期范围查询

## [v3.3.0] - 2026-06-28

### Added

  - 每条消息旁边添加收藏按钮（星标图标）
  - 收藏列表对话框，支持查看和移除收藏
  - 收藏数据持久化存储到IndexedDB
  - Toolbar新增"我的收藏"按钮

### Changed
- 版本号升级至v3.3.0（第三阶段功能增加）
